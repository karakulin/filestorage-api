## Install
* Install docker and docker-compose
* ```$ git clone https://github.com/karakulin/filestorage-api.git```
* ```$ cd filestorage-api```
* ```$ chmod 777 storage/```
* ```$ cp .env.example .env```
* ```$ docker-compose up```



#### MINIMAL DOCKER VERSION:
* Engine: 18.03+
* Compose: 1.21+
